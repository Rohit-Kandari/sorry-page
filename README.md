# Maafi Naama 🥺💗

Ek chota sa cute sorry page — sirf ek maksad ke liye, meri best friend ko manane ke liye 🎀

## Isme kya hai?

- **Sorry page** — ek pyara sa apology message, jahan "Maaf kiya" button na chahte hue bhi bada hota jaata hai 😅
- **Snack picker** — accept karne ke baad, sorry ke badle mein kuch treats choose karne ka option (Pulse Toffee, Melody Toffee, Lay's Blue, Ice-cream)
- **Share option** — jo choose kiya uska cute screenshot generate karke share/download kar sakte ho
